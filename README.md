# :floppy_disk: Installation
가상환경 생성
```
$ conda create -n venv-name python=3.10
```

가상환경 실행
```
$ conda activate venv-name
```

Git Clone
```
$ git clone git@github.com:mondayy1/Jambaram-DataServer.git
```

패키지 설치
```
$ pip install -r requirements.txt
```

서버 실행
```
$ uvicorn main:app --reload
```

localhost:8000 접속!
