# :floppy_disk: Installation
git clone
```
$ git clone git@github.com:mondayy1/Jambaram-DataServer.git
```

goto dir
```
$cd Jambaram-DataServer
```

install requirements
```
$ pip install -r requirements.txt
```

run server
```
$ uvicorn main:app --reload
```

localhost:8000 접속!
